字段查询

按 id / image_path / created_at 字段查询已索引的图片记录，不走向量相似度。 支持「快捷查询」和「条件组合查询」两种模式。

🆔 按主键查询

精确匹配 id，返回单条记录

📁 按图片路径精确查询

完全匹配 image_path 字段

🔍 按路径模糊查询 (LIKE %x%)

image_path 包含关键字（不分大小写取决于 Milvus 配置）

🕒 按入库时间范围

毫秒时间戳，留空表示不限。提示：可填入 Date.now() 当作「今天」

🧩 条件组合查询（AND 串联）

填一个或多个条件，留空字段不参与过滤。多个条件同时给定时用 AND 串联。 服务器最终会回显生成的 Milvus 表达式。

⚙️ 直接传 Milvus 表达式

注意：本模式不做白名单校验，expr 字符串直接交给 Milvus 执行。 适合 QueryCondition 表达不了的复杂查询（OR、子查询、JSON 字段等）。
